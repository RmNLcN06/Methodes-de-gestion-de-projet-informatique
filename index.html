Comment contribuer au projet?

C'est un long romaaaaaaaaaaan c'est une belle histoire

  • Télécharger Github Desktop sur le site officiel
  • Télécharger Visual Studio Code sur le site officiel
  • Ouvrir Github Desktop, connecter le compte
  • Aller dans "cloner un répertoire", selectionner "Lexique-collaboratif"
  • Cliquer sur "Ouvrir le projet dans Vs Code"

A chaque session de modification du projet:

  • !IMPORTANT! Vérifier via le bouton "Fetch Origin" que la version du projet en Local est bien la version la plus récente du projet.
  • Effectuer les modificationss voulues depuis VS Code, télécharger l'extension "live server" (facultatif) pour plus de confort de développement.
  • Une fois les modifications terminées, cliquer sur "Commit to main" (en bas à gauche de l'ecran sur Github Desktop) après avoir donné un nom aux modifications et les avoir décrites."
  • Cliquer sur "Push origin"
  • Bien joué! Modifications effectuées!
